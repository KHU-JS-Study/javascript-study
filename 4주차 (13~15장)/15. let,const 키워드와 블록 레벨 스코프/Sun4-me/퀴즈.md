# 📝 간단한 퀴즈

1. a, b, c의 값을 구하시오.
```js
var title = 'book';
console.log(title); // a

var title = 'movie';
console.log(title); // b

title = 'music';
console.log(title); // c
```
2. a, b, c의 값을 구하시오.
```js
let title = 'book';
console.log(title); // a

let title = 'movie';
console.log(title); // b

title = 'music';
console.log(title); // c
```

3. a, b, c의 값을 구하시오.
```js
const title = 'book';
console.log(title); // a

const title = 'movie';
console.log(title); // b

title = 'music';
console.log(title); // c
```
